用 CSS 画一朵花

2026 年 3 月 15 日 · 技术

不依赖图片，纯用 CSS 实现一朵简单的花。花瓣用 border-radius 弯出弧度，渐变上色，旋转排列。

思路

核心思路很简单：一个圆形作为花蕊，五个椭圆作为花瓣，每个花瓣旋转 72 度，就形成了一朵五瓣花。

花瓣

每个花瓣是一个 div，通过 border-radius: 50% 变成椭圆，加上径向渐变从粉到白：

  • 宽 40px，高 80px
  • border-radius: 50%
  • 渐变从 #f4a9b8 到 #fff
  • 定位到底部中心，绕底部中心旋转

花蕊

一个 30px 的圆，黄色渐变，定位在花的正中央。加一点 box-shadow 让它看起来更有质感。

茎和叶子

茎是一条细长的绿色矩形，加一点 border-radius 让边缘柔和。叶子用另一个椭圆，旋转 30 度贴在茎上，渐变从深绿到浅绿。

纯 CSS 作画本质上就是用几何形状的组合与变换来拼凑画面。限制反而带来创意。

最终效果

一朵粉色的五瓣花，绿色的茎和叶子，漂浮在白色背景上。很简单，但做完的那一刻还挺开心的。就像小时候用蜡笔画了一朵花，然后贴在冰箱上。
